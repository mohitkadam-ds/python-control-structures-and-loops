# Python Control Structures and Loops
This repository contains Python programs focused on conditional
statements and looping constructs. The aim is to strengthen
logic building and problem-solving skills.
## Topics Covered
- Nested if conditions
- if-elif based calculator
- for and while loops
- break, continue and pass statements
- Basic numerical problems
